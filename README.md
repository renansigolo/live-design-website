# Live Design

Source code of the Live Design website

## Usage

### Install Dependencies

```bash
npm i
```

### Development server

```bash
npm start
```

## Testing

```bash
npm run build:prod

npm run server
```

## Build and Deploy

- Staging

  `npm run deploy`

- Production

  `npm version <[major | minor | patch]>`
